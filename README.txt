# Fraction Escape Island

เกมเว็บการศึกษาเรื่องเรียงลำดับเศษส่วนจากน้อยไปมาก

## วิธีใช้
1. แตกไฟล์ ZIP
2. เปิด `index.html` ด้วยเบราว์เซอร์ เช่น Chrome, Safari, Edge
3. เล่นได้ทันที ไม่ต้องติดตั้งโปรแกรม

## ไฟล์
- index.html
- style.css
- script.js
- images/  สำหรับใส่รูปเพิ่มเติม
- sounds/  สำหรับใส่เสียงเพิ่มเติม

หมายเหตุ: เวอร์ชันนี้ใช้ภาพ/ไอคอนแบบ CSS และ Emoji เพื่อให้เปิดได้ทันทีโดยไม่ต้องพึ่งไฟล์ภาพภายนอก
